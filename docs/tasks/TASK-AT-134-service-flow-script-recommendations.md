## Objetivo unico
Melhorar a vinculacao entre Fluxos e Scriptoteca com recomendacoes por tag, categoria, canal e uso historico.

## Escopo
1. Sugerir scripts ao criar etapa.
2. Mostrar scripts mais usados em fluxos semelhantes.
3. Alertar script vencido ou obsoleto.
4. Permitir substituicao rapida.

## Acceptance Criteria
1. Admin encontra scripts relevantes sem vasculhar a lista.
2. Script obsoleto nao e recomendado.
3. Sugestoes nao criam vinculo automatico sem confirmacao.
