# AlwaysTrack - roteiro guiado de apresentacao

## Preparacao rapida
1. Rode `npm run demo:reset:local`.
2. Suba API e web.
3. Para exibir a faixa de guia no app, use `VITE_DEMO_MODE=true`.
4. Entre como `admin@example.com`.

## Abertura
Mensagem central:

> O AlwaysTrack transforma toda venda com nota em ranking confiavel, auditavel e acionavel. E transforma duvida operacional em FAQ/Wiki viva, consultavel e governada.

## Jornada principal

### 1. Central Operacional Hoje
- Mostrar o estado do dia.
- Clicar em um card de nota pendente.
- Frase-chave: "Nao e dashboard decorativo; cada card leva para uma acao real."

### 2. DANFE e revisao
- Mostrar a nota pendente.
- Abrir `Timeline`.
- Mostrar upload, extracao, auditoria e decisao.
- Frase-chave: "A nota nao desaparece em uma caixa preta; ela deixa rastro."

### 3. Ranking explicavel
- Abrir Ranking.
- Clicar em `Explicar` em um vendedor.
- Mostrar notas que entraram, total, quantidade, ticket medio e excecoes.
- Frase-chave: "Se alguem contestar o ranking, o sistema prova."

### 4. Extratos e exportacao
- Abrir Extratos.
- Mostrar consolidado por vendedor/grupo.
- Exportar CSV se fizer sentido.
- Frase-chave: "O ranking tem lastro financeiro e exportavel."

### 5. FAQ para Wiki
- Abrir FAQ resolvida/promovida.
- Abrir Wiki vinculada.
- Mostrar que a pergunta continua no FAQ, mas virou conhecimento validado.
- Frase-chave: "Conhecimento solto vira patrimonio consultavel."

### 6. Notificacoes, permissoes e auditoria
- Abrir sino de notificacoes.
- Abrir Configuracoes > Matriz de permissoes.
- Abrir Auditoria se houver tempo.
- Frase-chave: "O produto tem governanca, nao so telas."
